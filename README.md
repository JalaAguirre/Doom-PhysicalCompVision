# Doom-PhysicalCompVision
A controller to play Doom that integrates physical computing and computer vision

The following hardware and its controls:
1. Joystick - WASD
2. Body Threshold - Strafe Left and Right (comma and period)
3. Gesture Detection - Fire
4. Voice Detection - Use 
5. One Leg Detection - Run
6. Fingers detection - Weapon Selection

Finger Detection and Strafe Threshold Prerequisite:
- Python Version: 3.12.7
Type these commands in the terminal:
- & D:/Downloads/Doom-PhysicalCompVision/.venv/Scripts/Activate.ps1
- & "C:\Users\...\Programs\Python\Python312\python.exe" -m venv mp_compatible_env
- .\mp_compatible_env\Scripts\Activate.ps1
- pip install opencv-python mediapipe
